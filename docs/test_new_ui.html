💡 Quick Examples

Search Queries:

  • • health - Simple keyword
  • • "machine learning" - Exact phrase
  • • author:okonkwo - By author
  • • year:2020 - By year
  • • faculty:science - By faculty
  • • health AND education - Boolean AND
  • • title:cancer NOT lung - Boolean NOT

API Endpoints:

  • • /api/search/advanced?q=...
  • • /api/citations/<id>
  • • /api/author/<id>/metrics
  • • /api/citations/update/<id>
  • • /api/search/suggest?q=...
